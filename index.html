צריך לחבר Firebase פעם אחת

הקובץ כבר כמעט מוכן. הכנסתי לך את רוב פרטי הפרויקט, ונשאר רק למלא את databaseURL בתוך FIREBASE_CONFIG.

אחרי שממלאים את ההגדרות ומעלים את הקובץ ל־Firebase Hosting / Netlify / GitHub Pages, ההזמנה בוואטסאפ תהיה לחיצה אחת ותעבוד נכון.

  1. פתח Firebase Console וצור פרויקט חדש.
  2. הפעל Realtime Database במצב test.
  3. הוסף Web App והעתק את כתובת ה־Realtime Database.
  4. חפש בקובץ את PASTE_YOUR_DATABASE_URL והדבק שם את הכתובת האמיתית.
  5. אחסן את הקובץ אונליין ופתח את הכתובת משם, כדי שהקישור בוואטסאפ יהיה לחיץ.
